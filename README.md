# 🤖 Chatbot Project

A simple chatbot application built using **React**, **TypeScript**, and **Vite**. The chatbot can answer basic commands, perform simple fun tasks, and stores chat history in the browser.

## 🚀 Live Demo

https://chatbot-project-ts.onrender.com

## ✨ Features

* 💬 Interactive chatbot conversation
* 👋 Responds to greetings like **Hi** and **Hello**
* 🪙 **Flip a Coin** command
* 🎲 **Roll a Dice** command
* 📅 Displays the **current date**
* 💾 Saves chat history using **Local Storage**
* 🔄 Automatically scrolls to the latest message
* ⌨️ Press **Enter** to send a message
* ❌ Press **Escape** to clear the input field
* 🧹 Clear all chat messages
* ⏳ Loading spinner while waiting for a response

## 🛠️ Technologies Used

* React
* TypeScript
* Vite
* CSS
* Day.js
* SuperSimpleDev Chatbot Library

## ▶️ Run Locally

Clone the repository:
git clone https://github.com/rbharathi7995/chatbot-project.git

Go to the project folder:
cd chatbot-project

Install dependencies:
npm install

Start the development server:
npm run dev
